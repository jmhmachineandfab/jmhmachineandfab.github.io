## 🚀 Getting Started

To run this project locally, clone the repository and opening the index.html file.

### Prerequisites

*   A web browser (Chrome, Firefox, etc.)
*   Git installed on your machine

### Installation & Setup

1.  **Clone the repository** to your local machine:
    ```bash
    git clone https://github.com/jmhmachineandfab/jmhmachineandfab.github.io
    ```
   

2.  **Navigate** to the project directory:
    ```bash
    cd jmhmachineandfab.github.io
    ```

3.  **Open** the `index.html` file in your preferred web browser to view the project.
